# 📊 Sales Dashboard in Excel

## Overview

This project is a beginner-friendly interactive Sales Dashboard created using Microsoft Excel. It analyzes sample sales data and presents key business insights through Pivot Tables, Pivot Charts, and KPI metrics.

The goal of this project was to practice Excel data analysis skills and build a portfolio project for GitHub.

## Features

* 📈 Total Sales KPI
* 💰 Total Profit KPI
* 📊 Sales by Region (Column Chart)
* 📉 Monthly Sales Trend (Line Chart)
* 📋 Pivot Tables for data summarization
* 🎯 Clean and easy-to-read dashboard layout

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* SUM and AVERAGE formulas
* Excel Tables
* Basic dashboard design
